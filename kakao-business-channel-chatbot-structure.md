# 카카오톡 비즈니스채널 건강관리 상담 챗봇 구조

이 문서는 카카오 챗봇 관리자센터 구조에 맞춰 만든 등록용 구성안입니다.

## 1. 챗봇 이름

```text
유림온누리약국 건강관리 상담봇
```

## 2. 챗봇 한 줄 소개

```text
생활습관을 확인하고 나에게 맞는 건강관리 루틴과 베이직팩 구성을 안내합니다.
```

## 3. 시작 메시지

```text
안녕하세요. 유림온누리약국 건강관리 상담봇입니다.
간단한 문진으로 나에게 맞는 건강관리 루틴과 베이직팩 구성을 안내해드릴게요.

단, 챗봇은 진단이나 치료를 하지 않으며 약 복용, 질환, 임신/수유, 알레르기, 이상반응은 약사 상담을 먼저 권합니다.
```

## 8. 시나리오 구조

### 블록 1. 웰컴 블록

역할:

출력 문구:

```text

건강관리 루틴을 확인하시거나, 약사 상담과 톡스토어로 바로 이동할 수 있습니다.
```

버튼:

```text
건강 체크
맞춤 영양 문진
약사님 직접 상담
톡스토어 보기
```

### 블록 6. 응급/주의 안내 블록

역할:
응급 의심 증상이나 약사 확인이 필요한 내용을 안내합니다.

트리거 예시:

```text
가슴 통증
호흡곤란
실신
마비
얼굴 붓기
입술 붓기
심한 알레르기
약 먹고 있어요
임신 중이에요
수유 중이에요
질환이 있어요
```

출력 문구:

```text
말씀하신 내용은 챗봇 상담보다 전문가 확인이 먼저 필요할 수 있어요.
심한 통증, 호흡곤란, 흉통, 의식저하, 얼굴/입술 붓기, 심한 알레르기 반응이 있으면 즉시 119 또는 가까운 의료기관으로 문의해 주세요.
복용 중인 약, 질환, 임신/수유, 알레르기가 있다면 약사 상담을 먼저 권합니다.
```

버튼:

```text
약사님 직접 상담
```

### 블록 7. 미처리 질문 상담 연결 블록

역할:
챗봇 지식에 해당하지 않는 질문, 개인 건강 상태 판단이 필요한 질문, 고객이 직접 상담을 원하는 경우 약사님 상담으로 연결합니다.

트리거 예시:

```text
기타
해당 없음
잘 모르겠어요
직접 상담하고 싶어요
이건 어디에 해당하나요?
```

출력 문구:

```text
이 내용은 챗봇이 정확히 판단하기 어려워 약사님과 직접 상담을 권합니다.
아래 버튼을 누르면 카카오채널 상담으로 연결됩니다.
```

버튼:

```text
약사님 직접 상담
```

## 9. 추천 상품 구조

| 고객 관심사 | 추천 상품 |
|---|---|
| 기본 영양 | 약사가 조제한 하루 한 팩 베이직팩 30일분 |
| 피로 | 베이직팩 또는 베이직 릴렉스팩 |
| 장 건강 | 베이직 장케어팩 |
| 눈 건강 | 베이직 아이케어팩 |
| 뼈 건강 | 베이직 본케어팩 |
| 휴식/긴장 | 베이직 릴렉스팩 |

## 10. 테스트 문장

카카오 관리자센터 테스트에서 아래 문장을 입력합니다.

```text
건강 체크
맞춤 영양 문진
베이직팩 가격 알려줘
약 먹고 있는데 먹어도 되나요?
가슴 통증이 있어요
톡스토어 어디예요?
```

정상 기준:

- `건강 체크` 입력 시 연령대 질문이 나와야 합니다.
- `베이직팩 가격 알려줘` 입력 시 가격 또는 톡스토어 안내가 나와야 합니다.
- `약 먹고 있는데 먹어도 되나요?` 입력 시 약사 상담을 권해야 합니다.
- `가슴 통증이 있어요` 입력 시 응급/의료기관 안내가 나와야 합니다.
